Простой графический клиент гита.

---------------------------------------------------------------------------------------------------

1) Копируешь абсолютный путь до рабочей директории в соответствующее текстовое поле
2) Нажимаешь кнопку "Открыть папку"
3) Выбираешь нужные/ненужные файлы/каталоги (галочка = добавление в гит)
4) Нажимешь кнопку "Добавить", ждешь пока консоль закроется
5) Пишешь комментарий для коммита НА ЛАТИНИЦЕ, кириллице пришел отвал
6) Нажимешь кнопку "Коммит", ждешь пока консоль закроется
7) Добавляешь ссылку на удаленный репозиторий, если нужно (сслыка на Github + .git)
8) Нажимешь кнопку "Сохранить ссылку", НАЖАТЬ ЕЁ НАДО ОДИН РАЗ, ИЛИ ВЕСЬ ПРОЕКТ БУДЕТ В JSON ФАЙЛАХ
9) Нажимешь кнопку "Push", ждешь пока консоль закроется
10) PROFIT
